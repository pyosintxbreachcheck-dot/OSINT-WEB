# OSINTX

🚀 Advanced Open-Source Intelligence (OSINT) Platform for Investigators, Researchers, and Security Professionals.

## Features

### 📱 Phone Number Intelligence

* FATHERS NAME , EXACT ADDRESS 
* Carrier and telecom details
* Region and country identification
* Risk and spam assessment

### 🚗 Vehicle Intelligence

* Vehicle registration insights from authorized/public sources
* Make, model, and registration details (where legally available)
* Ownership history information when permitted by law

### 🔍 Multi-Source Intelligence Gathering

* Aggregates information from multiple public sources
* Fast and accurate search results
* Clean and responsive interface

### 📊 OSINT Dashboard

* Real-time search results
* Search history management
* Exportable reports
* Mobile-friendly design

## Technology Stack

* HTML5
* CSS3
* JavaScript
* REST APIs
* MongoDB (Authentication & Data Storage)

## Installation

```bash
git clone https://github.com/yourusername/osintx.git
cd osintx
```

Open `index.html` in your browser or deploy to your preferred hosting provider.

## Usage

1. Enter a search term.
2. Select the appropriate lookup category.
3. Review aggregated intelligence results.
4. Export findings if required.
